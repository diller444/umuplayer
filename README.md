# umuplayer
An alternative front-end for YouTube Music

## Features

- [x] Search
- [x] Audio
- [x] Channels
- [x] Playlists
- [x] Homepage
- [x] Uses piped.kavin.rocks proxy for images and audio
- [x] Open audio format (OPUS 160k)
- [ ] Videos
- [ ] Mixes
- [ ] Autoplay
- [ ] Settings
- [ ] Favicon
